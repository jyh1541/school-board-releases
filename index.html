실제 화면을
미리 만나보세요.

대시보드부터 시간표, 메모, 급식, 북마크, 업무자료, 연락처까지. 클릭하면 더 크게 볼 수 있어요.

01 / 대시보드

오늘 하루를 한눈에

시계, 급식, 시간표, 학사일정, 메모까지 — 위젯 기반 대시보드에서 오늘 필요한 모든 정보를 한 화면에.
02 / 시간표

NEIS 자동 시간표

학교 이름만 검색하면 시간표가 자동으로. API 키 없이 NEIS에서 바로.
03 / 메모

간편하게 기록하세요

떠오르는 생각, 할 일, 업무 메모를 빠르게 기록하고 관리하세요.
04 / 급식

오늘 뭐 먹지?

NEIS 연동으로 오늘의 급식 메뉴를 자동으로 확인. 알레르기 정보도 함께.
05 / 북마크

자주 쓰는 사이트, 빠르게

NEIS, 교육포털, 학교 홈페이지 등 자주 사용하는 웹사이트를 한 번에.
06 / 업무자료

학교 업무 문서를 깔끔하게

교재, 수업 자료, 공문 등 필요한 파일을 카테고리별로 정리하고 빠르게 검색.
07 / 연락처

교직원 연락처를 한 곳에

교장선생님, 교감선생님, 동료 교사들의 연락처를 한 곳에서 관리. 검색과 그룹 분류도 지원합니다.
