교무실의 모든 것

선생님이 학교에서 필요한 모든 도구를 하나의 위젯에 담았습니다.

대시보드

시계, 날씨, 시간표, 급식을 한 화면에. 벤토 그리드 레이아웃으로 위젯을 자유롭게 배치하세요.

드래그 배치 위젯 크기 조절

시간표

NEIS에서 자동으로 불러오거나 직접 입력하세요. 학반별, 교사별, 학생 개별 조회까지.

NEIS 연동 교과별 조회

급식 정보

조·중·석식 3끼 모두 확인. 알레르기 정보도 함께 표시되어 안심하고 식사하세요.

3식 제공 알레르기 표시

일정 & D-Day

학사일정과 개인 할 일을 통합 관리. D-Day 카운터로 중요한 날짜를 놓치지 마세요.

학사일정 연동 알림 설정

포스트잇 메모

자유 배치 보드에 드래그 앤 드롭. 리치 텍스트 편집, 6가지 파스텔 색상.

드래그 배치 리치 텍스트

연락처

교사·학생 연락처를 분리 관리. 엑셀 붙여넣기, CSV 가져오기, 인쇄 기능까지.

엑셀 연동 인쇄 지원

계산기

사칙연산부터 단위 변환, 등급 계산기까지. 배점 계산으로 시험 출제도 쉽게.

단위 변환 등급 계산

업무포털 자동 로그인

나이스, K-에듀파인 자동 로그인. 인증서 선택부터 비밀번호 입력까지 원클릭.

나이스 K-에듀파인

20+ 테마

벚꽃부터 네온까지 20종 프리셋. 원하는 색상으로 커스텀 테마도 만들 수 있어요.

다운로드 및 설치 시 안내

다운로드 시 경고가 뜨나요?

코드 서명 전이라 브라우저에서 보안 경고가 나타날 수 있으며, 실제 보안과는 무관합니다.

"확인되지 않은 파일 다운로드" 창이 나타나면 그대로 [확인되지 않은 파일 다운로드]를 눌러주세요.

"파일을 안전하게 다운로드하시겠습니까?" 창이 나타나면 [저장]을 눌러주세요.

설치할 때도 경고가 뜨나요?

설치 파일 실행 시 Windows 보안 경고가 나타날 수 있습니다. 아래 순서대로 조치하시면 됩니다.

1 "추가 정보"를 클릭하세요
2 하단의 "실행"을 클릭하면 설치가 시작됩니다
