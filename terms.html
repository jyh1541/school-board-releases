스쿨보드(School Board)

이용약관

최종 수정일: 2026년 4월 10일

제1조 (목적)

본 이용약관(이하 "본 약관")은 "스쿨보드(School Board)" 애플리케이션(이하 "본 앱")의 이용 조건 및 절차, 개발자와 사용자 간의 권리/의무 및 책임 사항을 규정합니다.

제2조 (정의)

  1. "본 앱"이라 함은 개발자가 제공하는 Windows 데스크톱 애플리케이션 "스쿨보드(School Board)"를 말합니다.
  2. "사용자"라 함은 본 앱을 설치하여 사용하는 자를 말합니다.
  3. "서비스"라 함은 본 앱이 제공하는 모든 기능(시간표 관리, 급식 정보, 학사일정, 날씨, 메모, 캘린더, 연락처, 계산기, 업무자료, 북마크, 자동 로그인 등)을 말합니다.
  4. "라이선스"라 함은 본 앱의 정식 사용 권한을 말합니다.

제3조 (약관의 효력 및 변경)

  1. 본 약관은 본 앱 설치 시 사용자가 동의하는 것으로 효력이 발생합니다.
  2. 개발자는 관련 법령을 위배하지 않는 범위에서 본 약관을 변경할 수 있습니다.
  3. 약관 변경 시 앱 내 공지사항 또는 업데이트 안내를 통해 고지합니다.

제4조 (서비스의 내용)

본 앱은 다음과 같은 서비스를 제공합니다.

  1. 대시보드: 위젯 형태의 개인화된 업무 화면 (시계, 날씨, 캘린더, D-Day, 메모, 할 일, 시간표, 급식 등)
  2. 시간표 관리: 교사용/학반별/학생 개별/교과별 시간표 작성 및 관리
  3. 캘린더 및 할 일: 월간/주간 캘린더, 할 일 CRUD, 반복 일정, 알림
  4. 메모 (포스트잇 보드): 자유 배치 메모, 투두리스트 메모
  5. 급식 정보: NEIS API 연동 급식 정보 및 알레르기 정보
  6. 학사일정: NEIS API 연동 학사일정 조회
  7. 날씨 정보: 실시간 날씨 및 대기질 정보
  8. 연락처 관리: 교사/학생 연락처 관리, 엑셀 내보내기/인쇄
  9. 계산기: 사칙연산, 급식비, 배점, 등급 계산
  10. 업무자료: 구글 시트 연동 업무 자료 관리
  11. 북마크: 자주 사용하는 웹사이트 바로가기
  12. 확성기/오늘의 알림: 구글 시트 연동 학교 알림
  13. 업무포털 자동 로그인: 나이스, K-에듀파인 자동 로그인
  14. 데이터 백업/복원: 로컬 자동 백업, 수동 내보내기/불러오기
  15. 기타: D-Day 카운터, 테마 설정, 위젯 모드, 자동 업데이트

제5조 (라이선스)

  1. 본 앱은 라이선스 키 기반 인증을 통해 사용 권한을 부여합니다.
  2. 라이선스 종류는 다음과 같습니다.
    • 단일 기기 라이선스(Single): 1개의 기기에서만 사용 가능
    • 기타 개발자가 정하는 라이선스 유형
  3. 라이선스 인증 시 기기 식별자(해시 처리됨)와 라이선스 키가 서버로 전송됩니다.
  4. 인증 토큰은 발급일로부터 14일간 유효하며, 자동 갱신됩니다.
  5. 오프라인 환경에서는 마지막 인증일로부터 14일 이내에 한하여 앱 사용이 가능합니다.
  6. 사용자는 라이선스 키를 제3자에게 양도, 공유, 대여할 수 없습니다.
  7. 부정 사용이 확인된 경우 개발자는 라이선스를 취소할 수 있습니다.

제6조 (이용 제한)

다음의 경우 개발자는 서비스 이용을 제한할 수 있습니다.

  1. 라이선스 키를 불법으로 복제, 배포, 공유하는 경우
  2. 본 앱을 역공학, 디컴파일, 리버스 엔지니어링하는 경우
  3. 본 앱의 정상적인 운영을 방해하는 경우
  4. 기타 관련 법령에 위배되는 방식으로 사용하는 경우

제7조 (저작권)

  1. 본 앱의 모든 저작권은 개발자에게 있습니다.
  2. 사용자는 본 앱을 정당한 라이선스 취득 후 개인적/업무적 목적으로만 사용할 수 있습니다.
  3. 본 앱의 복제, 배포, 변조, 판매 등은 금지됩니다.

제8조 (제3자 서비스)

  1. 본 앱은 다음 제3자 서비스를 연동하여 기능을 제공합니다.
    • NEIS 교육정보개방포털: 급식, 학사일정, 시간표, 학교 정보
    • Open-Meteo: 날씨 정보
    • 네이버: 날씨 정보
    • Google Sheets: 업무 알림 데이터
    • GitHub: 자동 업데이트
    • Supabase: 라이선스 인증
    • 나이스, K-에듀파인: 업무포털 자동 로그인 (사용자 인증 정보 필요)
  2. 제3자 서비스의 변경, 중단 등으로 인해 본 앱의 일부 기능이 제한될 수 있으며, 이에 대해 개발자는 책임을 지지 않습니다.
  3. 제3자 서비스의 이용약관 및 개인정보처리방침은 각 서비스의 규정을 따릅니다.

제9조 (면책 조항)

  1. 서비스 중단: 천재지변, 시스템 장애, 제3자 서비스 중단 등 불가항력적인 사유로 인해 서비스가 중단된 경우 개발자는 책임을 지지 않습니다.
  2. 데이터 손실: 사용자의 기기 오류, 악성코드, 부주의한 조작 등으로 인한 데이터 손실에 대해 개발자는 책임을 지지 않습니다. 단, 본 앱은 자동 백업 기능을 제공합니다.
  3. 제3자 서비스 정보: NEIS, 날씨 API 등 제3자 서비스에서 제공되는 정보의 정확성에 대해서는 개발자가 보증하지 않습니다.
  4. 업무포털 자동 로그인: 해당 기능은 공인인증서를 이용한 자동화 편의 기능이며, 업무포털의 정책 변경 등으로 인해 작동하지 않을 수 있습니다.
  5. 자동 로그인 보안: 업무포털 로그인 정보는 사용자 PC에 AES-256으로 암호화되어 저장되나, 기기 자체의 보안 취약점으로 인한 정보 유출에 대해서는 개발자가 책임을 지지 않습니다.

제10조 (데이터의 소유권)

  1. 사용자가 본 앱에 입력한 모든 데이터(시간표, 메모, 연락처, 할 일 등)의 소유권은 사용자에게 있습니다.
  2. 사용자는 언제든지 데이터를 내보내기(.board, .xlsx) 할 수 있습니다.
  3. 사용자가 앱을 삭제하면 로컬에 저장된 모든 데이터가 함께 삭제됩니다.

제11조 (업데이트 및 유지보수)

  1. 본 앱은 GitHub Releases를 통해 자동 업데이트를 제공합니다.
  2. 사용자는 업데이트를 통해 최신 기능 및 보안 패치를 받을 수 있습니다.
  3. 업데이트 시 기존 데이터는 자동으로 보존됩니다.
  4. 대규모 구조 변경 시 데이터 마이그레이션이 자동으로 수행됩니다.
  5. 업데이트 다운로드 시 자동 백업이 수행됩니다.

제12조 (분쟁 해결)

  1. 본 약관과 관련하여 분쟁이 발생한 경우, 개발자와 사용자는 상호 협의하여 해결합니다.
  2. 협의로 해결되지 않는 경우, 관련 법령에 따른 절차를 따릅니다.
  3. 준거법은 대한민국 법률로 합니다.
