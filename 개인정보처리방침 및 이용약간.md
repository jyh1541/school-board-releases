# 개인정보처리방침

**최종 수정일:** 2026년 4월 10일

---

"학교 근무 도우미(School Board)"(이하 "본 앱")는 사용자의 개인정보를 소중히 여기며, 관련 법령을 준수합니다. 본 개인정보처리방침은 본 앱이 어떤 정보를 수집하고, 어떻게 사용하며, 어떻게 보호하는지 설명합니다.

---

## 1. 개인정보 수집 항목 및 방법

### 1-1. 자동으로 수집되는 정보

| 수집 항목 | 목적 | 저장 위치 |
| --- | --- | --- |
| 기기 식별자(Device ID) | 라이선스 기기 바인딩 | Supabase 서버 (해시 처리됨) |

- **기기 식별자**는 Windows 운영체제의 Machine GUID 값을 SHA-256 알고리즘으로 해시하여 생성되므로, 원래 기기 정보를 역추적할 수 없습니다.

### 1-2. 사용자가 직접 입력하는 정보

| 수집 항목 | 목적 | 저장 위치 |
| --- | --- | --- |
| 라이선스 키 | 소프트웨어 사용 인증 | Supabase 서버 (인증 시에만 전송) |
| 학교 정보 (학교명, 학교코드) | NEIS 급식·학사일정·시간표 조회 | 사용자 PC (로컬 저장) |
| 날씨 지역 정보 | 날씨 정보 조회 | 사용자 PC (로컬 저장) |
| 교사·학생 연락처 | 개인 일정 및 연락처 관리 | 사용자 PC (로컬 저장) |
| 시간표, 메모, 할 일, D-Day 등 | 개인 업무 관리 | 사용자 PC (로컬 저장) |
| 업무포털 로그인 정보 | 나이스·K-에듀파인 자동 로그인 | 사용자 PC (Electron safeStorage AES-256 암호화) |
| 구글 시트 URL | 확성기·오늘의 알림 데이터 조회 | 사용자 PC (로컬 저장) |

### 1-3. 수집하지 않는 정보

본 앱은 다음 정보를 **수집하지 않습니다.**

- 사용자의 이름, 주민등록번호, 이메일 주소, 전화번호
- 사용자의 정확한 위치 정보(GPS 등)
- 사용자의 사진, 영상, 음성
- 웹 브라우징 기록
- 기기에 저장된 다른 앱의 데이터

---

## 2. 개인정보의 이용 목적

수집한 정보는 다음 목적으로만 사용됩니다.

1. **라이선스 인증 및 관리**: 정품 사용자 확인, 기기 바인딩, 인증 갱신
2. **교육행정정보 제공**: NEIS Open API를 통한 급식, 학사일정, 시간표 정보 조회
3. **날씨 정보 제공**: Open-Meteo, 네이버 날씨 API를 통한 날씨 및 대기질 정보 조회
4. **업무포털 자동 로그인**: 사용자가 저장한 공인인증서 정보로 나이스·K-에듀파인 접속
5. **업무 알림**: 구글 시트를 통한 확성기/오늘의 알림 데이터 표시
6. **앱 자동 업데이트**: GitHub Releases를 통한 최신 버전 확인 및 다운로드

---

## 3. 개인정보의 보관 및 파기

### 3-1. 보관 기간

| 정보 | 보관 기간 | 보관 위치 |
| --- | --- | --- |
| 기기 식별자 (해시) | 라이선스 유효 기간 중 | Supabase 서버 |
| 라이선스 인증 토큰 | 발급일로부터 14일 | 사용자 PC (localStorage) |
| 사용자 데이터 (시간표, 메모 등) | 앱 사용 기간 중 | 사용자 PC (로컬 파일) |
| 업무포털 로그인 정보 | 앱 사용 기간 중 | 사용자 PC (암호화 저장) |

### 3-2. 파기 방법

- **사용자 PC의 데이터**: 앱 내 "데이터 내보내기" 기능으로 백업 후, 앱 삭제 시 `%APPDATA%/SchoolBoard` 폴더가 함께 삭제됩니다.
- **서버 데이터**: 라이선스 만료 또는 해지 시 관련 기기 식별자 정보는 서버에서 삭제됩니다.
- **라이선스 토큰**: 14일 경과 후 자동 만료되며, 재인증 시 기존 토큰은 교체됩니다.

## 4. 개인정보의 제3자 제공

본 앱은 사용자의 개인정보를 외부에 **판매, 대여, 거래하지 않습니다.**

단, 다음의 경우 서비스 제공에 필요한 최소한의 정보가 외부 서버로 전송됩니다.

| 제공받는 자 | 전송 정보 | 목적 | 전송 방식 |
| --- | --- | --- | --- |
| Supabase (Supabase Inc.) | 기기 식별자(해시), 라이선스 키 | 라이선스 인증·갱신 | HTTPS 암호화 통신 |
| NEIS 교육정보개방포털 | 학교코드, 연도, 월, 일 | 급식·일정·시간표 조회 | HTTPS 공개 API |
| Open-Meteo | 위도·경도 (지역 기반) | 날씨 정보 조회 | HTTPS 공개 API |
| 네이버 | 지역코드 | 날씨 정보 조회 | HTTPS 암호화 통신 |
| Google Sheets (gviz) | 시트 URL | 업무 알림 데이터 조회 | HTTPS 공개 API |
| GitHub (github.com) | 버전 정보 | 자동 업데이트 확인 | HTTPS 공개 API |

- 업무포털(나이스, K-에듀파인) 자동 로그인 시 입력된 공인인증서 정보는 **사용자 PC 내에서만 암호화되어 저장**되며, 외부 서버로 전송되지 않습니다.
- 모든 통신은 **HTTPS(TLS 1.2 이상)**로 암호화되어 전송됩니다.

---

## 5. 개인정보의 안전성 확보 조치

본 앱은 사용자의 개인정보를 보호하기 위해 다음과 같은 조치를 취하고 있습니다.

1. **데이터 암호화**: 업무포털 로그인 정보는 Electron safeStorage(AES-256)로 암호화하여 저장합니다.
2. **기기 식별자 해시 처리**: 기기 고유 정보는 SHA-256으로 해시 처리되어 원본을 복원할 수 없습니다.
3. **통신 구간 암호화**: 모든 외부 API 통신은 HTTPS로 암호화됩니다.
4. **로컬 데이터 저장**: 사용자의 모든 업무 데이터(시간표, 메모, 연락처 등)는 외부 서버가 아닌 사용자 PC에만 저장됩니다.
5. **자동 백업 암호화**: 자동 백업 파일은 사용자 PC의 로컬 디렉토리에 저장되며, 외부로 전송되지 않습니다.

---

## 6. 사용자의 권리

사용자는 언제든지 다음과 같은 권리를 행사할 수 있습니다.

1. **데이터 열람 및 수정**: 앱 내에서 모든 개인 데이터를 열람하고 수정할 수 있습니다.
2. **데이터 내보내기**: 설정 > 데이터 관리에서 모든 데이터를 `.board` 또는 `.xlsx` 형식으로 내보낼 수 있습니다.
3. **데이터 삭제**: 앱 내에서 개별 데이터를 삭제하거나, 앱 제거 시 전체 데이터를 삭제할 수 있습니다.
4. **라이선스 해지**: 라이선스 만료 또는 앱 삭제를 통해 서버에 저장된 기기 식별자 정보의 삭제를 요청할 수 있습니다.

## 7. 14세 미만 아동의 개인정보 보호

본 앱은 학교 선생님을 대상으로 하는 업무 보조 도구이며, 14세 미만 아동으로부터 개인정보를 직접 수집하지 않습니다. 학생 연락처 등의 정보는 사용자(선생님)가 직접 입력하는 것으로, 본 앱은 이를 로컬에만 저장합니다.

## 제4조 (서비스의 내용)

본 앱은 다음과 같은 서비스를 제공합니다.

1. **대시보드**: 위젯 형태의 개인화된 업무 화면 (시계, 날씨, 캘린더, D-Day, 메모, 할 일, 시간표, 급식 등)
2. **시간표 관리**: 교사용/학반별/학생 개별/교과별 시간표 작성 및 관리
3. **캘린더 및 할 일**: 월간/주간 캘린더, 할 일 CRUD, 반복 일정, 알림
4. **메모 (포스트잇 보드)**: 자유 배치 메모, 투두리스트 메모
5. **급식 정보**: NEIS API 연동 급식 정보 및 알레르기 정보
6. **학사일정**: NEIS API 연동 학사일정 조회
7. **날씨 정보**: 실시간 날씨 및 대기질 정보
8. **연락처 관리**: 교사/학생 연락처 관리, 엑셀 내보내기/인쇄
9. **계산기**: 사칙연산, 급식비, 배점, 등급 계산
10. **업무자료**: 구글 시트 연동 업무 자료 관리
11. **북마크**: 자주 사용하는 웹사이트 바로가기
12. **확성기/오늘의 알림**: 구글 시트 연동 학교 알림
13. **업무포털 자동 로그인**: 나이스, K-에듀파인 자동 로그인
14. **데이터 백업/복원**: 로컬 자동 백업, 수동 내보내기/불러오기
15. **기타**: D-Day 카운터, 테마 설정, 위젯 모드, 자동 업데이트

## 제5조 (라이선스)

1. 본 앱은 라이선스 키 기반 인증을 통해 사용 권한을 부여합니다.
2. 라이선스 종류는 다음과 같습니다.
   - **단일 기기 라이선스(Single)**: 1개의 기기에서만 사용 가능
   - 기타 개발자가 정하는 라이선스 유형
3. 라이선스 인증 시 기기 식별자(해시 처리됨)와 라이선스 키가 서버로 전송됩니다.
4. 인증 토큰은 발급일로부터 14일간 유효하며, 자동 갱신됩니다.
5. 오프라인 환경에서는 마지막 인증일로부터 14일 이내에 한하여 앱 사용이 가능합니다.
6. 사용자는 라이선스 키를 제3자에게 양도, 공유, 대여할 수 없습니다.
7. 부정 사용이 확인된 경우 개발자는 라이선스를 취소할 수 있습니다.

## 제8조 (제3자 서비스)

1. 본 앱은 다음 제3자 서비스를 연동하여 기능을 제공합니다.
   - **NEIS 교육정보개방포털**: 급식, 학사일정, 시간표, 학교 정보
   - **Open-Meteo**: 날씨 정보
   - **네이버**: 날씨 정보
   - **Google Sheets**: 업무 알림 데이터
   - **GitHub**: 자동 업데이트

   - **나이스, K-에듀파인**: 업무포털 자동 로그인 (사용자 인증 정보 필요)
2. 제3자 서비스의 변경, 중단 등으로 인해 본 앱의 일부 기능이 제한될 수 있으며, 이에 대해 개발자는 책임을 지지 않습니다.
3. 제3자 서비스의 이용약관 및 개인정보처리방침은 각 서비스의 규정을 따릅니다.

---

## 제9조 (면책 조항)

1. **서비스 중단**: 천재지변, 시스템 장애, 제3자 서비스 중단 등 불가항력적인 사유로 인해 서비스가 중단된 경우 개발자는 책임을 지지 않습니다.
2. **데이터 손실**: 사용자의 기기 오류, 악성코드, 부주의한 조작 등으로 인한 데이터 손실에 대해 개발자는 책임을 지지 않습니다. 단, 본 앱은 자동 백업 기능을 제공합니다.
3. **제3자 서비스 정보**: NEIS, 날씨 API 등 제3자 서비스에서 제공되는 정보의 정확성에 대해서는 개발자가 보증하지 않습니다.
4. **업무포털 자동 로그인**: 해당 기능은 공인인증서를 이용한 자동화 편의 기능이며, 업무포털의 정책 변경 등으로 인해 작동하지 않을 수 있습니다.
5. **자동 로그인 보안**: 업무포털 로그인 정보는 사용자 PC에 AES-256으로 암호화되어 저장되나, 기기 자체의 보안 취약점으로 인한 정보 유출에 대해서는 개발자가 책임을 지지 않습니다.
